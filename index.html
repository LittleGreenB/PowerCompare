1 · Upload usage data

Drop your Usage.csv here or click to choose a file

Format: column 1 = Date (YYYYMMDD), then 48 half-hourly columns (0:00 … 23:30) of kWh.

2 · Pricing plans

How it works: each 30-minute interval is placed into a rate band by its start time. Time windows may cross midnight (pick a "from" later than the "to", e.g. 23:00 → 11:00). Tiers like "First 25 kWh/day" apply to that band's total usage each day. There is no controlled-load column in the usage file, so controlled-load rates are ignored.
